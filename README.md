# FUTURE_PE_02 - AI-Powered Website with AI Chatbot

## Task Description
This project was created as part of the **Future Interns Prompt Engineering Internship – Task 2**.

The objective of Task 2 was to build an **AI-powered website** using modern AI tools for:
- AI-generated website content
- AI-assisted layout/design
- AI chatbot or FAQ assistant

---

## AI Tools Used

- **10Web AI (WordPress)** – AI-powered website generation (layout + content)
- **WordPress CMS** – Website management
- **Tidio Lyro AI Chatbot** – AI chatbot trained on website content

---

## How AI Was Used

### 1. AI Website Creation
- The website structure, sections, and written content were generated using **10Web AI** based on prompts describing an AI-powered study planner.
- AI assisted in generating:
  - Hero section
  - Feature descriptions
  - Pricing layout
  - FAQ content

### 2. AI Chatbot Integration
- **Tidio Lyro AI chatbot** was integrated via the WordPress plugin.
- The chatbot automatically learned from the website content and answers user queries such as:
  - What the platform does
  - Features offered
  - How users can get started
- The chatbot provides real-time AI-powered assistance to visitors.
